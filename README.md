# Um Estudo sobre Computação Quântica

## Visão Geral

Este repositório contém:

1. Exemplos de algoritmos quânticos
2. Monografia (.tex)
3. Apresentação (.tex)
4. Monografia e Apresentação no formato pdf.

## Sobre a monografia e apresentação

A monografia e a apresentação estão em formato `tex` os pdfs finais podem ser encontrados em `/pdfs`. Os arquivos originais estão nas pastas `monografia` e `apresentacao` respectivamente.

## Sobre os Exemplos

Os exemplos estão em `exemplos/` para executá-los inicie um *virtual environment* na raíz do repositório:

```sh
uv venv
source .venv/bin/activate
```

e instale as dependências:

```sh
uv pip install -r requirements.txt
```

por fim, inicialize o kernel do notebook python como o *virtual environment* criado.

Duvidas sobre instalação e uso do `uv` consulte a documentação oficial: https://docs.astral.sh/uv.
